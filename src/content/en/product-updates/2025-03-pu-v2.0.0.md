Greetings, Applivery users! We're thrilled to bring you our March 2025 update, packed with powerful enhancements that give you more control over your mobile ecosystem. From improved app signing detection to smarter security controls and flexible user management, this release is all about making your workflow smoother and more efficient. Let's explore what's new!

**Quick notes before we start:**

- Development signing type enables developers to install and test apps directly on registered devices
- TestFlight signing type facilitates beta distribution through Apple's TestFlight platform for internal and external testing
- The ticket tracking feature builds on the existing chat-based support tool introduced in previous months
- The Unassigned role provides flexible access control ideal for gradual user onboarding before assigning specific permissions
- Public publications can be disabled in Enterprise Store > Customize > Security > Allow Public Publications setting
